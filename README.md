# Cubway Surfers

### How to start play

1. open - [deploy's link](https://cubway-surfers.web.app/welcome)
2. click - "Let's go!"

### Installation

Cubway Surfers requires [Node.js](https://nodejs.org/) v12+ to run.

Install the dependencies.

```sh
$ cd rs-clone
$ npm install
```

Run server.

```sh
$ ng serve
```
